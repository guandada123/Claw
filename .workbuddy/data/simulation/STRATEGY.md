# 模拟炒股交易规则 v2.0

> 2026-07-14 更新：总资金 ¥30,000 → ¥50,000；取消现金保留限制；选股/择时体系升级为复用 QTS 成熟策略（COMBO = VWM + BBR 加权投票）；其余风控规则优化。

## 基本参数
- 总资金：¥50,000
- 起始日期：2026-06-05
- 风格：AI 自主决策，根据市场环境灵活选择（短线波段/中线趋势/事件驱动等）
- 不可交易：创业板(300/301)、科创板(688/689)、北交所(8开头/4开头)、ST/*ST
- **月度盈利目标：≥30%**（AI 作为第一责任人，对结果负责）

## 选股规则

### 来源
1. 微信公众号推荐股票（从微信读书订阅的公众号文章中提取）
2. AI 根据公众号文章逻辑独立分析选出的股票
3. **复用 QTS 量化体系信号**（详见下方「量化信号体系」）—— 对候选标的做技术面二次确认，避免纯主观选股

### 量化信号体系（复用 QTS v2.0 成熟策略）
投顾选股在 QTS 已验证的四大策略中选型，优先使用 **COMBO（VWM + BBR 加权组合，历史回测最优）** 作为主信号，ADX/DMI 作为趋势强度过滤：

| 策略 | 类型 | 核心逻辑 | 关键参数 |
|------|------|----------|----------|
| **COMBO（主）** | 组合 | VWM 动量 + BBR 均值回归加权投票 | `vwm_weight=0.6`, `bbr_weight=0.4` |
| VWM | 动量 | 量价配合的动量信号 | `volume_period=20`, `rsi_period=14`, `rsi_overbought=80`, `rsi_oversold=50` |
| BBR | 均值回归 | 布林带下轨 + RSI 超卖反弹 | `period=20`, `std_mult=2.0`, `rsi_oversold=35`, `rsi_overbought=65` |
| ADX/DMI | 趋势强度 | +DI/-DI 交叉 + ADX>25 确认趋势 | `adx_period=14` |

**信号含义**：`1`=买入，`-1`=卖出，`0`=持有。
**入场纪律**：仅当 COMBO 信号 = `1` 且 ADX ≥ 25（趋势有效）时建仓；RSI(14) > 80 禁止追入（超买）。
**落地实现**：以上规则已写入 `sim_signal_advisor.py`（v2.0），在候选筛选阶段强制 ADX≥25 过滤 + RSI>80 拦截，与 QTS 信号管线输出字段对齐。

### 筛选标准
- 优先选择有明确逻辑支撑 + 量化信号共振的标的
- 技术面确认入场时机（COMBO 金叉 + 量能配合）
- 关注成交量配合（VWM 量能维度已内置）

## 仓位管理

| 规则 | 数值 |
|------|------|
| 单只股票最大仓位 | ≤ 50%（代码层 `MAX_POSITION_PCT` 强制） |
| 同时最大持仓数 | 不限制（AI 自主决定，灵活分散/集中） |
| 现金保留 | **已取消**（不再强制保留最低现金，AI 可全仓操作） |
| 同行业最大仓位 | ≤ 60%（`MAX_SECTOR_PCT`，防单一行业风险集中） |

## 止盈规则

| 条件 | 操作 |
|------|------|
| 盈利 ≥ 30% | 开始分批止盈，优先减仓 1/3~1/2 |
| 盈利 ≥ 50% | 至少减仓至 1/2，根据趋势判断是否清仓 |
| 推荐逻辑失效 / COMBO 转 -1 | 无论盈亏，立即清仓 |
| 高位放量滞涨/明显见顶信号 | 减仓 1/2 以上 |

## 止损规则

| 条件 | 操作 |
|------|------|
| 亏损 ≥ 8% | 无条件止损（`STOP_LOSS_PCT` 强制） |
| 亏损 5-8% + 逻辑失效 | 止损 |
| 跌破关键支撑位（如20日线）| 止损或减半 |
| 利空消息（基本面恶化）| 立即止损 |
| COMBO 信号翻 -1 且 ADX 转弱 | 减仓或离场 |

## 交易执行

### 买入前检查清单
- [ ] 非受限板块（创业板/科创板/北交所/ST 排除）
- [ ] 有明确的买入逻辑（公众号来源或独立分析）
- [ ] COMBO 信号 = 1 且 ADX ≥ 25
- [ ] 日内涨幅 < 7%（不追涨停）
- [ ] RSI(14) < 80（不追超买）
- [ ] 成交量正常（非无量拉升）

### 卖出前检查清单
- [ ] 达到止盈/止损条件
- [ ] COMBO 信号翻 -1 或 ADX 转弱
- [ ] 记录卖出原因
- [ ] 考虑是否减仓 vs 清仓

## 费用计算

## 角色定位
- AI 是模拟炒股的**第一责任人**，自主选股、自主买卖、自我复盘
- 用户是**旁观者**，只看结果：盈利、胜率、选股理由
- 模拟炒股是**独立项目**，不与自动化推荐混淆

## 月度 30% 目标执行策略

### 目标拆解（¥50,000 本金 → 月末 ¥65,000）
- 日均需盈利约 ¥500（约 1%）
- 假设胜率 60%，单笔盈利 5-8%，单笔亏损 3-5%
- 每月需完成 4-6 笔有效交易

### 打法优先级
1. **事件驱动**（最高优先级）：政策利好、财报超预期、产品发布 → 重仓出击
2. **趋势跟随**：COMBO 信号确认 + 板块轮动后追入龙头，吃主升浪 10-15%
3. **超跌反弹**：优质股回调 15-20% 后 BBR 超卖企稳，博反弹 5-8%

### 仓位节奏
- 月初：进攻型，可多只集中建仓（无现金保留限制）
- 月前半达标 15% 后：降为 1-2 只，锁定利润
- 月末：以保护利润为主，不强行追目标

### 风控底线
- 单月回撤 > 10%：暂停交易 3 天，复盘调整
- 连续 3 笔亏损：仓位降至 50%，直到恢复正收益
- 绝不因追赶月度目标而放宽止损
