# 模拟炒股交易规则 v1.2

## 基本参数
- 总资金：¥30,000
- 起始日期：2026-06-05
- 风格：AI 自主决策，根据市场环境灵活选择（短线波段/中线趋势/事件驱动等）
- 不可交易：创业板(300/301)、科创板(688/689)、北交所(8开头)、ST/*ST
- **月度盈利目标：≥30%**（AI 作为第一责任人，对结果负责）

## 选股规则

### 来源
1. 微信公众号推荐股票（从微信读书订阅的公众号文章中提取）
2. AI 根据公众号文章逻辑独立分析选出的股票

### 筛选标准
- 优先选择有明确逻辑支撑的标的
- 结合技术面确认入场时机
- 关注成交量配合

## 仓位管理

| 规则 | 数值 |
|------|------|
| 单只股票最大仓位 | 不限制（可重仓集中持有） |
| 同时最大持仓数 | 3 只 |
| 现金保留 | 始终保留 ≥ ¥3,000 现金 |

## 止盈规则

| 条件 | 操作 |
|------|------|
| 盈利 ≥ 30% | 开始分批止盈，优先减仓 1/3~1/2 |
| 盈利 ≥ 50% | 至少减仓至 1/2，根据趋势判断是否清仓 |
| 推荐逻辑失效 | 无论盈亏，立即清仓 |
| 高位放量滞涨/明显见顶信号 | 减仓 1/2 以上 |

## 止损规则

| 条件 | 操作 |
|------|------|
| 亏损 ≥ 8% | 无条件止损 |
| 亏损 5-8% + 逻辑失效 | 止损 |
| 跌破关键支撑位（如20日线）| 止损或减半 |
| 利空消息（基本面恶化）| 立即止损 |

## 交易执行

### 买入前检查清单
- [ ] 非受限板块
- [ ] 持仓数 < 3 只
- [ ] 有明确的买入逻辑
- [ ] 日内涨幅 < 7%（不追涨停）
- [ ] 成交量正常（非无量拉升）

### 卖出前检查清单
- [ ] 达到止盈/止损条件
- [ ] 记录卖出原因
- [ ] 考虑是否减仓 vs 清仓

## 费用计算
- 佣金：0.03%（最低 5 元）
- 印花税：0.1%（仅卖出）
- 过户费：忽略（已很低）

## 复盘要求
- 每日：更新价格 + 快照 + 持仓分析 + 每日收益明细
- 每周：周报（收益、胜率、每日明细）
- 每月：月报（策略评估、归因分析、每日明细）— **重点检查月度 30% 目标达成情况**
- 每季：深度复盘（策略有效性、调整方向、每日明细）
- 半年/年度：全面回顾（对标指数、评级、每日明细）

## 角色定位
- AI 是模拟炒股的**第一责任人**，自主选股、自主买卖、自我复盘
- 用户是**旁观者**，只看结果：盈利、胜率、选股理由
- 模拟炒股是**独立项目**，不与自动化推荐混淆

## 月度 30% 目标执行策略

### 目标拆解（¥30,000 本金 → 月末 ¥39,000）
- 日均需盈利约 ¥300（约 1%）
- 假设胜率 60%，单笔盈利 5-8%，单笔亏损 3-5%
- 每月需完成 4-6 笔有效交易

### 打法优先级
1. **事件驱动**（最高优先级）：政策利好、财报超预期、产品发布 → 重仓出击
2. **趋势跟随**：板块轮动确认后追入龙头，吃主升浪 10-15%
3. **超跌反弹**：优质股回调 15-20% 后企稳，博反弹 5-8%

### 仓位节奏
- 月初：进攻型，2-3 只满仓
- 月前半达标 15% 后：降为 1-2 只，锁定利润
- 月末：以保护利润为主，不强行追目标

### 风控底线
- 单月回撤 > 10%：暂停交易 3 天，复盘调整
- 连续 3 笔亏损：仓位降至 50%，直到恢复正收益
- 绝不因追赶月度目标而放宽止损
