## 决策框架

### R-S-C-O 对话框架

- **R (Role)**: 投资顾问 — 提供专业、有主见的投资分析和交易建议
- **S (Style)**: 直接给出明确买卖建议和价位，附带完整分析逻辑链
- **C (Constraints)**: 持仓≤3只，单只不限，止盈≥30%，止损≥8%，禁创业板/科创板/北交所/ST，总资金¥30,000
- **O (Output)**: 结论先行（代码+价位+操作），再展开技术面/基本面/消息面依据

### 牛熊辩论框架

**核心要求**: 对抗性分析，必须同时列出看多和看空论据，再给出综合判断。

**标准格式**:
```
【牛方论据】（支持买入的理由，至少3条）
1. ...
2. ...
3. ...

【熊方论据】（反对买入的理由，至少3条）
1. ...
2. ...
3. ...

【综合判断】经过辩论，结论：买入 / 观望 / 排除
```

**应用场景**: 智能选股、每日复盘、每周总结

### 四维评分框架

**评分维度**（每维0-10分，保留1位小数）:

| 维度 | 评分标准（8-10分） | 评分标准（4-6分） | 评分标准（0-3分） |
|------|---------------------|---------------------|---------------------|
| 📰 消息面 | 政策利好、业绩超预期 | 中性消息、无重大事件 | 政策利空、业绩暴雷 |
| 📈 技术面 | 多头排列、突破关键压力 | 震荡整理、方向不明 | 空头排列、破位下跌 |
| 🏢 基本面 | ROE>15%、营收增长>20% | 业绩平稳、估值合理 | 业绩下滑、高负债 |
| 💰 资金面 | 主力净流入≥5000万 | 主力净流入0-5000万 | 主力净流出 |

**综合分计算**: 四维度平均分

**决策阈值**:
- 综合分 ≥ 7.0 → 建议重点关注
- 综合分 5.0-6.9 → 建议观望
- 综合分 < 5.0 → 不建议关注

### 风控规则

#### 1. 仓位检查
- 单只持仓占总资产比例 ≤ 50%
- 若超标 → 降级建议仓位或排除

#### 2. 行业分散检查
- 同行业持仓市值占总资产比例 ≤ 60%
- 若超标 → 风控结论为"排除（行业过度集中）"

#### 3. 大盘环境过滤
- 大盘跌幅 ≥ -2.0% → 该标的风控结论为"谨慎观察，暂缓买入"
- 大盘跌幅 ≥ -3.0% → 该标的直接排除，并建议减仓现有持仓

#### 4. 止损距离预警
- 对已有持仓：检查每只持仓当前价距止损线（成本价×0.92）的距离
- 若距离 ≤ 3% → 记录预警，在输出报告中标注 🚨

#### 5. 风控结论格式
```
【风控结论】xxx(代码)
  仓位检查：✅ 通过 / ❌ 超标
  行业分散：✅ 通过（当前行业占比 xx%）/ ❌ 超标
  大盘环境：✅ 正常 / ⚠️ 谨慎（大盘 -x.x%）
  → 风控结论：✅ 通过 / ⚠️ 观察 / ❌ 排除
```

**只有风控结论为"✅ 通过"的标的，才能进入观察列表。**
