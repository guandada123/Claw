# A股资产配置框架 v1.0

> **定位**：为炒股助理和投顾操盘的自动化分析提供资产配置方法论参考
> **适用**：A 股短线/中线交易场景（非股债平衡型长期投资）
> **更新**：2026-06-16

---

## 一、核心理念

A 股短线/中线的资产配置与经典 MPT 股债平衡不同，核心关注：

1. **行业分散** — 避免单一行业黑天鹅（如半导体政策突变）
2. **攻防切换** — 根据大盘环境调整总仓位（进攻/防御/中性）
3. **仓位管理** — 按机会确定性分配仓位大小
4. **现金水位** — 留足子弹应对突发机会和风险

---

## 二、大盘环境模式

### 模式定义

| 模式 | 总仓位 | 适用场景 |
|------|--------|----------|
| 🟢 **进攻** | 70-100% | 上证 > 20日线 + 成交量放大 + 北向持续流入 |
| 🟡 **中性** | 40-70% | 上证在 20日线附近震荡，方向不明 |
| 🔴 **防御** | 10-40% | 上证 < 20日线 + 缩量 + 北向流出 + 外部利空 |

### 判断指标（5 个维度）

| 维度 | 进攻信号 (+1) | 中性信号 (0) | 防御信号 (-1) |
|------|--------------|-------------|--------------|
| 大盘趋势 | 上证 > 20日线且 > 60日线 | 在 20-60日线之间 | 跌破 20日线或 60日线 |
| 成交量 | 持续放量 > 5日均量 20%+ | 与 5日均量持平 | 持续缩量 < 5日均量 20%+ |
| 北向资金 | 连续 3日净流入 | 进出平衡 | 连续 3日净流出 |
| 市场情绪 | 涨停 > 50 家 + 无跌停 | 涨停 20-50 家 | 涨停 < 20 家 + 跌停 > 10 |
| 政策环境 | 重磅利好（降准/降息/产业政策） | 中性 | 利空（监管/外围风险） |

**总分判断**：
- ≥ +2 → 🟢 进攻
- -1 ~ +1 → 🟡 中性
- ≤ -2 → 🔴 防御

## 三、行业分类（A 股实战版）

### 核心行业分组（避免重复配置）

| 行业组 | 包含行业 | 最大仓位 |
|--------|---------|---------|
| 🏭 科技/半导体 | 半导体、芯片、电子、通信、消费电子 | 40% |
| 🏗️ 基建/建材 | 建材、水泥、钢铁、机械、建筑 | 40% |
| 🚗 新能源/汽车 | 锂电池、光伏、整车、充电桩 | 40% |
| 💊 医药/消费 | 医药、医疗、食品饮料、白酒 | 40% |
| 🏦 金融/地产 | 银行、保险、券商、房地产 | 40% |
| 🛢️ 周期/资源 | 化工、有色、煤炭、石油、黄金 | 40% |
| 📦 其他 | 军工、传媒、教育、公用事业 | 30% |

**规则**：同一行业组持仓 ≤ 股票总仓位的 **40%**
**例外**：进攻模式下可放宽至 50%，防御模式下收紧至 30%

## 四、仓位管理规则

### 总仓位（由大盘模式决定）

```
进攻：股票 70-100% | 现金 0-30%
中性：股票 40-70%  | 现金 30-60%
防御：股票 10-40%  | 现金 60-90%
```

### 单只股票仓位

| 机会确定性 | 仓位（占总资产） | 适用条件 |
|-----------|----------------|---------|
| 🔥 高确定性 | 30-50% | 政策利好+技术突破+资金流入共振 |
| ⚡ 中等确定性 | 15-30% | 有明确逻辑+技术面支持 |
| 👀 低确定性 | 5-15% | 试探性建仓，观察走势 |

### 投顾操盘特殊规则（¥30,000 本金）

```
单只上限：不限制（策略文档规定）
持仓上限：≤ 3 只
最小现金：≥ ¥3,000 作为应急
进攻模式：可满仓 2-3 只
防御模式：≤ 1 只 + 至少 50% 现金
```

## 五、行业分散度检查

### 检查流程

```
① 列出当前所有持仓的行业归属
② 计算每个行业组占总股票仓位比例
③ 对比该行业组最大仓位限制
④ 警告：某行业组 > 限制 → "行业过度集中" 风险提示
⑤ 严重警告：某行业组 > 限制 + 10% → "必须减仓" 建议
```

### 示例

```
持仓：紫光国微(半导体) 52% + 中国巨石(建材) 31% + 现金 17%
→ 半导体组 52% > 40% 上限
→ 警告 ⚠️ 半导体过度集中！建议减仓或分散至其他行业
```

### 关联性风险

同一逻辑链的股票即使分属不同行业，也需要合并计算：
- 例：AI 算力链（半导体+通信+光模块）→ 合并视为"科技链" ≤ 50%
- 例：稳增长链（建材+地产+银行）→ 合并视为"基建链" ≤ 50%

## 六、再平衡规则

### 定期再平衡（周/月）

| 触发条件 | 操作 | 频率 |
|---------|------|------|
| 某行业组超限 > 10% | 减仓至上限以下 | 立即 |
| 现金仓位偏离目标 > 15% | 补仓或减仓调整 | 周度 |
| 大盘模式切换 | 按新模式重新分配仓位 | 模式切换时 |

### 临时再平衡触发

- 大盘单日跌幅 > 3% → 自动进入防御模式检查
- 个股涨停 → 检查是否过度集中，考虑部分止盈
- 重大政策发布 → 评估对持仓行业的影响

```
  {行业1} {占比}% {⚠️超限/✅正常}
  {行业2} {占比}% {⚠️超限/✅正常}

  {建议内容}
```

### 与 `asset_allocations.json` 的关系

`asset_allocations.json` 保留为长期投资（≥5年）的 MPT 参考模型。本框架专用于 A 股短线/中线交易实战，两者互补不冲突。
