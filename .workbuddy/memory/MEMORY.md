## Marvis + WorkBuddy 桥接系统（2026-06-10 搭建）

### 定位
- **WorkBuddy = 调度方（大脑）**：分析/决策/生成/自动化调度
- **Marvis = 执行方（手脚）**：系统操控/文件管理/浏览器自动化/数据采集
- 关键限制：Marvis 无 API/SDK 暴露，联动走共享文件系统

### 桥接目录
`~/workbuddy_marvis_bridge/`
- `claw/` — A股投资系统任务+结果+原始数据
- `quant/` — 量化交易系统任务+结果+原始数据
- `shared/` — 共享行情缓存/配置
- `scripts/` — task_validator.py + file_watcher.sh + template.json
- `status/bridge.json` — 桥接状态

### 核心组件
- Skill: `~/.workbuddy/skills/marvis-bridge/` — 桥接调度技能
- 自动化: `Marvis Bridge Monitor` — 每3分钟扫描，ID: automation-1781023179949
- 文件监听: `file_watcher.sh` — fswatch 实时监听（需手动启动）
- 飞书通知: oc_9ee5303497f5e0e71666b610d6bdc346

### 任务格式
JSON 文件写入 `{project}/tasks/`，必填 task_id/project/type/source/title
完成归档到 `{project}/done/`，结果写入 `{project}/results/`
